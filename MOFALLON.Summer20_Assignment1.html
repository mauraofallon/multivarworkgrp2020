Examine variance differences between cities (Levene Test)

Levene test - uses this to evaluate homogeneity of variances assumption. Null hypothesis is that variances are equal (i.e., differences between variances is zero). Nonsignificant p-values mean that variances are roughly = and assumption has not been violated.

Must activate car package first.

library(car)
levene.spell <- leveneTest(SpellTestLong$`spelling test score`, SpellTestLong$city, 
    center = mean)
levene.spell  #highly significant! p<2.2e-16 
## Levene's Test for Homogeneity of Variance (center = mean)
##         Df F value    Pr(>F)    
## group    1  579.59 < 2.2e-16 ***
##       1498                      
## ---
## Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1

Highly significant p value. We reject null hypo that variances are equal. We can’t really assume equal variance for these data.

Run a t.test comparing Phila to Pitt, add an effect size (cohen d). Don’t assume equal variance

Must activate effsize package

IndTtest <- t.test(SpellTestLong$`spelling test score` ~ SpellTestLong$city, data = SpellTestLong)
IndTtest  # t=41.504, p< 2.2e-16, highly significant // Philly test scores higher than Pitt scores
## 
##  Welch Two Sample t-test
##

library(effsize)  #activate effsize package
CohenDSpelling <- cohen.d(SpellTestLong$`spelling test score`, SpellTestLong$city, 
    pooled = TRUE, paired = FALSE, na.rm = FALSE, mu = 0, hedges.correction = FALSE, 
    conf.level = 0.95, noncentral = FALSE, within = FALSE)

Summarize your finding

To compare spelling test scores between students at Temple and Pitt, an independent sample t-test was completed. Test results indicated a highly significant difference in test scores (t(886.82)= 41.504, p<2.2e-16, d=2.14), such that students from Temple (M=90.04, SD=4.78) achieved higher scores than students from Pitt (M=65.19, SD=15.68). The data do not support Dr. Dickey’s hypothesis.
